Constructor & Destructor Documentation

template<class T >
prism::memory::SharedPointer< T >::SharedPointer ( )

Constructs an empty null shared pointer instance. The managed pointer is set to 0.

template<class T >
prism::memory::SharedPointer< T >::SharedPointer ( T *  data)

Constructs a shared pointer that points to the data pointer. The data pointer becomes managed by this shared pointer and so must not be passed to another SharedPointer instance or deleted outside of this object.

template<class T >
prism::memory::SharedPointer< T >::SharedPointer ( const SharedPointer< T > &  copy)

Constructs a shared pointer that is a copy of copy. The new SharedPointer instance will point to the pointer that copy points to and the internal reference counter is incremented.

template<class T >
prism::memory::SharedPointer< T >::~SharedPointer ( )

Destroys this SharedPointer instance. The internal reference count for the managed pointer is decremented and if this SharedPointer instance held the last reference to the managed pointer then the managed pointer is also deleted.

Member Function Documentation

template<class T >
void prism::memory::SharedPointer< T >::clear ( )

Resets this SharedPointer instance (i.e. isNull() == true). The managed pointer is set to 0. The internal reference count for the managed pointer is decremented and if this SharedPointer instance held the last reference to the managed pointer then the managed pointer is also deleted.

template<class T >
T * prism::memory::SharedPointer< T >::data ( ) const

Returns the managed pointer. Note: do not delete or pass this pointer to another SharedPointer as the SharedPointer instances will become corrupt.

template<class T >
const bool prism::memory::SharedPointer< T >::isNull ( ) const

Returns true if the managed pointer is set to 0, false if otherwise it points to a valid object.

template<class T >
const bool prism::memory::SharedPointer< T >::isUnique ( ) const

Returns true if this SharedPointer instance is the only object that holds a reference to the managed pointer, false otherwise. Note: null pointers are never unique as they don't point to anything.

template<class T >
prism::memory::SharedPointer< T >::operator bool ( ) const

Returns true if this SharedPointer instance is not null, false otherwise. It can be used in if statements: if (shared_pointer) { ... }

template<class T >
const bool prism::memory::SharedPointer< T >::operator! ( ) const

Returns true if this SharedPointer instance is null, false otherwise. It can be used in if statements: if (!shared_pointer) { ... }

template<class T >
T & prism::memory::SharedPointer< T >::operator* ( ) const

Overloads the * operator so that it returns a reference to the value that the managed pointer points to.

template<class T >
T * prism::memory::SharedPointer< T >::operator-> ( ) const

Overloads the -> operator so that it returns the managed pointer.

template<class T >
SharedPointer< T > & prism::memory::SharedPointer< T >::operator= ( const SharedPointer< T > &  rhs)

This SharedPointer instance takes ownership of the managed pointer in rhs and the internal reference counter is incremented.

template<class T >
const int prism::memory::SharedPointer< T >::refCount ( ) const

Returns the number of SharedPointer objects that have ownership of the managed pointer.
