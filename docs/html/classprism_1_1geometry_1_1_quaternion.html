Quaternion prism::geometry::Quaternion::conjugate ( ) const

Returns a new Quaternion which is the result of negating the vector component of this quaternion. This quaternion remains unaltered. Note: when a quaternion has been normalised (has unit length of 1) the conjugate and inverse are identical. The quaternion needs to be of unit length in order to perform rotations correctly.

const float prism::geometry::Quaternion::dotProduct ( const Quaternion &  q1,
const Quaternion &  q2 
)
static

Returns a scalar which is the dot product of the two quaternions in the range [-1, 1]. The dot product of two quaternions represents the cosine of half of the angle needed to rotate one quaternion into the other. The larger the absolute value the result is (the closer it is to 1) means the closer the two quaternions are in orientation.

Quaternion prism::geometry::Quaternion::inverse ( ) const

Returns the inverse of this quaternion. The inverse is defined as dividing the conjugate by the magnitude. Note: when a quaternion has been normalised (has unit length of 1) the conjugate and inverse are identical. The quaternion needs to be of unit length in order to perform rotations correctly.
