8  * Queue is a container adapter that is based on the FIFO (first in first out) principle. It allows items to be added only at
9  * the end of the queue and items can only be removed from the start of the queue. A Queue has no iterators.
10  * It uses a List as the underlying container by default and basically forwards all function calls onto the list. Actually any container
11  * can be used as the underlying container by passing it along at construction i.e. Queue<int, vector<int> >, as long as
12  * the following functions are defined:
13  * empty()
14  * size()
15  * front()
16  * back()
17  * push_back()
18  * pop_front()
19  */
